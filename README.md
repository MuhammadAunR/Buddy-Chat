# Buddy Chat
A simple and friendly AI chatbot built with React and Gemini API. Chat with your AI buddy anytime!

---

## ✨ Features

- 💬 Real-time AI chat powered by Groq API (Llama 3.3)
- 🗂️ Conversation history saved to localStorage
- 📋 Copy any message to clipboard
- 🌙 Dark theme UI with smooth animations
- 📱 Responsive layout
- 🔁 Auto-expanding textarea input
- ⌨️ Send messages with `Enter` key
- 🗑️ Delete conversations from sidebar
- 📂 View previous conversations from sidebar

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + Vite |
| Styling | Tailwind CSS |
| AI API | Groq API (llama-3.3-70b) |
| Icons | Lucide React |
| Markdown | react-markdown |
| State Management | React Context API |
